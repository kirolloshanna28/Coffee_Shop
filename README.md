# Coffee_Shop
# Coffee Shop Sales Prediction

# Project Overview
This project analyzes coffee shop sales data and builds a predictive model to forecast the most demanded drinks.  
The goal is to clean and preprocess the data, explore sales patterns, and develop a machine learning model that helps in predicting future drink demand to support better decision-making.

# Project Steps

### 1. Data Cleaning
- Removed missing and inconsistent values
- Standardized column names
- Converted dates and times to proper formats

### 2. Data Preprocessing
- Feature engineering for date and time
- Grouping sales by day and drink type
- Preparing data for modeling

### 3. Data Visualization
Explored patterns in:
- Daily sales trends
- Most popular drinks
- Peak ordering times

### 4. Machine Learning Model
Built a predictive model to:
- Forecast drink demand
- Identify which drinks are most likely to be ordered

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

# Project Goal
Help coffee shop owners:
- Understand customer behavior
- Predict popular drinks
- Improve inventory planning
- Support data-driven decisions
